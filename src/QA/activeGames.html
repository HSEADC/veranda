Волейбол

Для игры в волейбол на даче вам понадобится волейбольный мяч, сетка и площадка. Разделите игроков на две команды и поставьте их по разные стороны сетки. Цель игры — передать мяч через сетку на сторону соперника так, чтобы он упал на поле противника. Команда, которая не смогла вернуть мяч, теряет очко.

Играем в крокет с друзьями

  1. Подготовка. Для игры в крокет на даче вам понадобится набор крокета, состоящий из деревянных молотков, шариков и арок.
  2. Правила игры. Разделите игроков на команды или играйте каждый за себя.
  3. Цель игры. Пройти по полю, попадая мячом через арки с помощью молотка за минимальное количество ходов.
